# GINIT
GINIT solves for zeros of mathematical equations by blending three complementary approaches:

- **Bisection**: to have the best initial guess possible.
- **Newton-Raphson**: a way of finding the zeros of polynomials.
- **Inverse**: a way of finding the zeros of non-continuous equations (often including trigs and logarithm).

## Features
- Interactive CLI/GUI interface with real-time plotting
- Web version for easy access

## Tools
- **Core**: Python, Numpy
- **CLI/GUI**: tkinter, matplotlib
- **Web App**: Flask, Tailwind CSS
